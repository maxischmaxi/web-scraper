# Web Scraper

A simple web scraper that extracts data from a website and saves it to a MONGODB database.

## Features

- Scrapes data from a specified URL
- Follows all links on the page
- Saves the scraped data to a MongoDB database

## Requirements

- Golang 1.24.5
- MongoDB
